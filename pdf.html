Deskripsi singkat / jabatan

Gunakan salah satu tombol di bawah untuk mengunduh atau membuka file PDF resume. Letakkan file resume.pdf pada folder yang sama dengan file HTML ini, atau ganti path pada link/JS sesuai lokasi file Anda.

⬇️ Download (link langsung) 🔍 Buka di tab baru

Catatan teknis: jika file PDF berada di domain lain dan server tidak mengizinkan CORS, tombol force-download JS mungkin gagal. Jika ingin file di-embed ke HTML (bukan link eksternal), pertimbangkan meng-upload PDF ke server yang Anda kontrol atau menggunakan base64 (untuk file sangat kecil saja).
